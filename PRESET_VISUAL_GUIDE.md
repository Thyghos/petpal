## Real User Scenarios

### Scenario 1: Sarah - New Puppy Owner
**Situation:** Just adopted 8-week-old Golden Retriever

**Choice:** ⭐ Essential Only

**Why:**
- Needs reminders for vet appointments
- Tracking puppy vaccinations (Health History)
- Frequent trips to vet/training (Travel Mode)

**Outcome:** Simple interface, not overwhelmed, focuses on puppy basics

---

### Scenario 2: Mike - Senior Dog Parent
**Situation:** 13-year-old Lab with arthritis & diabetes

**Choice:** 🏥 Medical Focus

**Why:**
- Multiple medications to track (Health History)
- Frequent vet visits & test results (Documents)
- Medical emergency info ready (Emergency QR)
- Managing insurance claims (Insurance)

**Outcome:** All medical tools at fingertips, streamlined care management

---

### Scenario 3: Lisa - Adventure Enthusiast
**Situation:** Travels every weekend with Border Collie

**Choice:** ✈️ Travel Ready

**Why:**
- Camping/hiking trip planner (Travel Mode)
- Emergency info if dog gets lost (Emergency QR)
- Vaccination proof for dog parks (Documents)

**Outcome:** Always prepared for adventures, quick access to travel essentials

---

### Scenario 4: Tom - Minimalist
**Situation:** Likes clean interfaces, uses few features

**Choice:** 👁️‍🗨️ Hide All → Shows only Reminders & Emergency QR

**Why:**
- Only needs appointment tracking
- Wants emergency info accessible
- Everything else is clutter

**Outcome:** Ultra-clean home screen, exactly what he needs

---

## Preset Switching Example

**Week 1:** Normal routine
- Using: ⭐ Essential Only
- Home shows: Reminders, Health, Travel

**Week 2:** Pet gets sick
- Switches to: 🏥 Medical Focus  
- Home shows: Health, Documents, Emergency, Insurance
- Manages illness efficiently

**Week 3:** Pet recovers
- Switches back to: ⭐ Essential Only
- Returns to normal routine

**Week 4:** Planning vacation
- Switches to: ✈️ Travel Ready
- Home shows: Travel, Emergency, Documents
- Prepares for trip

**One tap = New context!** 🎯

## Visual Design Elements

### Preset Buttons in Settings

```
┌─────────────────────────────────────┐
│ ⭐ Essential Only          ✓       │ ← Active
│    Reminders, Health, Travel        │
│    [Purple accent]                  │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│ 🏥 Medical Focus                    │ ← Inactive
│    Health, Docs, Emergency...       │
│    [Pink accent]                    │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│ ✈️  Travel Ready                    │ ← Inactive
│    Travel, Emergency, Docs          │
│    [Blue accent]                    │
└─────────────────────────────────────┘
```

### Color Coding

- **Essential** → Purple (⭐ favorite/important)
- **Medical** → Pink (💗 health/care)
- **Travel** → Blue (🌊 adventure/sky)
- **Show All** → Green (✅ complete)
- **Hide All** → Orange (⚠️ caution)
